# IBM Data Analyst Professional Certificate

Welcome to the **IBM Data Analyst Professional Certificate** learning repository.

This repository contains my notes, labs, assignments, practice exercises, and projects completed throughout the **IBM Data Analyst Professional Certificate** offered on **Coursera**.

The goal of this repository is to document my learning journey, reinforce key concepts, and build a portfolio of practical data analytics skills.

---

## 📖 About the Professional Certificate

The **IBM Data Analyst Professional Certificate** is a comprehensive program designed to prepare learners for entry-level data analyst roles.

The specialization consists of **11 courses** that progressively develop both theoretical knowledge and hands-on technical skills.

Throughout the program, learners gain experience with:

- Data Analytics fundamentals
- Excel for Data Analysis
- SQL and Relational Databases
- Python for Data Analysis
- Data Visualization
- Business Intelligence
- Data Collection and Wrangling
- Data Analysis using Python libraries
- Dashboard Development
- Capstone Projects
- Career preparation

The learning path follows a progressive approach where each course builds upon concepts introduced in previous courses.
